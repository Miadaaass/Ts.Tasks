Проект реализован на Python + FastApi. 
Задачи хранятся в массиве памяти.
Для запуска нужно открыть проект и в консоли прописать 2 команды 
"pip install fastapi uvicorn pydantic" далее "uvicorn main:app --reload --port 3000"
После перейти в любой удобный браузер и в поисковой строке прописать "http://localhost:3000/docs" в данном окне создаются задачи. 
Для проверки существующих задач открыть окно "http://localhost:3000/tasks".
